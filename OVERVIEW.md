## 🚀 快速开始

### 方法1: 使用启动脚本 (推荐)

**Windows:**
```bash
双击运行 start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

### 方法2: 手动启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 打开浏览器访问
# http://localhost:3000
```
