# 任务管理系统 (OneCar Task Manager)

一个简洁、美观的任务管理系统,支持任务的创建、查看、修改、删除、归档等核心功能。

## 功能特性

### 核心功能
- ✅ **任务管理**: 创建、查看、编辑、删除任务
- ✅ **任务归档**: 归档和取消归档任务
- ✅ **多维筛选**: 支持关键词搜索、状态、优先级、负责人筛选
- ✅ **任务详情**: 查看完整的任务信息
- ✅ **负责人管理**: 支持多负责人分配

### 任务属性
- 任务标题 (必填, 1-200字符)
- 任务描述 (可选, 最多1000字符)
- 任务状态: 待处理、进行中、已完成、已取消
- 优先级: 低、中、高、紧急
- 负责人: 支持多人
- 完成时间: 可选日期

### 用户体验
- 🎨 简洁美观的界面设计
- 🔒 危险操作二次确认机制
- 📱 响应式布局,支持多设备
- 💾 本地存储,数据持久化
- 🎯 直观的悬浮操作按钮

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5
- **构建工具**: Vite 5
- **日期处理**: Day.js
- **状态管理**: React Hooks
- **数据存储**: LocalStorage

## 使用说明

### 创建任务
1. 点击右下角的悬浮按钮
2. 点击"+"图标打开创建表单
3. 填写任务信息(标题、描述、负责人、优先级、完成时间)
4. 点击"创建"按钮

### 查看和编辑任务
1. 在任务列表中点击任务项
2. 查看任务完整信息
3. 点击"编辑"按钮可修改负责人和优先级

### 筛选任务
- **关键词搜索**: 在搜索框输入关键词,支持标题和描述搜索
- **状态筛选**: 选择待处理、进行中、已完成、已取消
- **优先级筛选**: 选择低、中、高、紧急
- **负责人筛选**: 选择一个或多个负责人

### 管理任务
1. 在列表中点击选择任务
2. 使用右下角悬浮按钮:
   - **删除**: 删除选中的任务(需确认)
   - **归档**: 归档选中的任务(需确认)
   - **取消归档**: 取消归档选中的任务(需确认)

### 查看归档任务
- 点击顶部"已归档任务"标签页查看所有归档的任务

## 设计特点

### 视觉设计
- 主色调: 蓝色系 (#1890ff)
- 优先级颜色:
  - 低: 灰色
  - 中: 蓝色
  - 高: 橙色
  - 紧急: 红色
- 圆角: 4-8像素
- 间距: 统一的 12-24像素间距

### 交互设计
- 任务项悬停效果
- 流畅的过渡动画
- 清晰的视觉反馈
- 友好的错误提示

### 表单验证
- 任务标题: 必填,1-200字符
- 任务描述: 可选,最多1000字符
- 负责人: 至少选择一个
- 优先级: 必选
- 完成时间: 不能早于当前日期

## 数据存储

系统使用浏览器的 LocalStorage 进行数据持久化存储:
- 任务数据: `onecar_tasks`
- 负责人数据: `onecar_assignees`

## 开发指南

### 添加新功能
1. 在 `src/types/task.ts` 中定义相关类型
2. 在 `src/services/taskService.ts` 中添加服务方法
3. 在 `src/components/` 中创建或修改组件
4. 在 `src/App.tsx` 中集成新功能

### 代码规范
- 使用 TypeScript 进行类型检查
- 遵循 ESLint 规则
- 组件使用函数式组件和 Hooks
- 保持组件的单一职责
